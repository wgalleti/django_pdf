## Requires

Python 3.6

WeasyPrint

xhtml2pdf


## Running

`git clone https://github.com/wgalleti/django_pdf.git`

`cd django_pdf`

`mkdir tmp`

`pipenv sync -d`

`pipenv shell`

`python manage.py makemigrations & python manage.py migrate`

`python manage.py runserver`
